# 🚀 365 Challenge Tracker

A modern and feature-rich **Challenge Tracking Application** built with **React.js** and **Firebase Firestore** that helps users build consistency by tracking daily goals, monitoring progress, maintaining streaks, and visualizing performance through analytics.

---

## 📸 Features

- ✅ Add, Edit & Delete Challenges
- 🎯 Custom Target Days
- 🔥 Real Streak Tracking
- 📅 One Click Per Day Validation
- 📝 Daily Notes & History
- 🏆 Achievement & Badge System
- 🎉 Confetti Celebration
- 🏅 Trophy Screen on Completion
- 📊 Analytics Dashboard
- 📈 Progress Visualization (Recharts)
- 📄 Global & Individual PDF Export
- 🔍 Search Challenges
- 🌙 Dark / ☀️ Light Theme
- 💾 LocalStorage Persistence
- ☁️ Firebase Firestore Cloud Sync
- 📱 Fully Responsive UI

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/shivamgupta12345/Challenge-Yourself.git
```

Move into project directory

```bash
cd Challenge Yourself
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
